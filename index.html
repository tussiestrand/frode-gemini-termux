Gemini CLI på Termux

Kjør Googles kraftige Gemini AI direkte fra terminalen på Android – ingen PC nødvendig.

Oppdatert mars 2026 • Termux + Node.js + @google/gemini-cli
Steg-for-steg installasjon
1

Installer Termux fra F-Droid

Last ned Termux fra F-Droid (ikke Play Store – den er utdatert). Åpne appen og oppdater pakkelisten.

# Oppdater pakkelisten
pkg update && pkg upgrade -y
2

Installer Node.js og npm

Gemini CLI krever Node.js versjon 18 eller nyere.

pkg install nodejs -y
# Verifiser installasjonen
node --version
npm --version
3

Installer Gemini CLI

Installer pakken globalt via npm så den er tilgjengelig fra hele systemet.

npm install -g @google/gemini-cli
# Bekreft at den er installert
gemini --version
4

Sett opp Gemini API-nøkkel

Gjør nøkkelen permanent tilgjengelig ved å legge den til i shell-konfigurasjonen.

# Hent nøkkel gratis på aistudio.google.com
echo 'export GEMINI_API_KEY="din-api-nokkel-her"' >> ~/.bashrc
source ~/.bashrc
Gratis nøkkel: Gå til aistudio.google.com og opprett en gratis Gemini API-nøkkel.
5

Start Gemini CLI

Nå er alt klart! Start en interaktiv samtale med Gemini.

gemini
# eller spesifiser en modell
gemini --model gemini-2.0-flash
Showcase – Hva kan du bruke Gemini CLI til?
🧠
Spill AI-assistent
Få hjelp til å løse problemer, forklare kode, eller brainstorme ideer direkte i terminalen.
gemini "Forklar hva rekursjon er"
💾
Analyser filer
Send filer direkte til Gemini for analyse, oppsummering eller refaktorering av kode.
gemini -i kode.py "Forbedre denne koden"
🌐
Oversett tekst
Oversett dokumenter, meldinger eller kode-kommentarer til et’n hvert språk.
gemini "Oversett til engelsk: [tekst]"
📊
Skriv skript
La Gemini skrive Bash- eller Python-skript for automatisering av Termux-oppgaver.
gemini "Lag et bash-skript som..."
🔍
Feilsjøking
Lim inn feilmeldinger og få umiddelbare løsningsforslag fra Gemini.
gemini "Hva betyr denne feilen: [error]"
📝
Oppretthold notater
Bruk Gemini til å oppsummere, formatere og organisere notater og tekster.
gemini -i notater.txt "Lag en oppsummering"
Tips og triks
💡

Bruk alias for rask tilgang

# Legg til i ~/.bashrc for egendefinerte snarveier
alias ai='gemini'
alias explain='gemini "Forklar på norsk: "'
📦

Pipe inn output fra andre kommandoer

# Send output direkte til Gemini
cat error.log | gemini "Hva er galt her?"
ls -la | gemini "Analyser disse filene"
🎨

Interaktiv modus med kontekst

# Start en lång samtale med kontekst
gemini --model gemini-2.5-pro
# Inne i chatten:
> Les denne filen: [filnavn]
> Lag en README basert på koden
Obs: Gratis API-nøkkel har bruksgrenser (ca. 1500 forespørsler/dag). For tung bruk anbefales Google AI Studio Pro-plan.
Modeller tilgjengelig: gemini-2.0-flash (rask), gemini-2.5-pro (kraftig), gemini-1.5-flash (lett)
